اموزش تایپ ده انگشتی 

یک برنامه‌ی دسکتاپ برای آموزش تایپ ده انگشتی فارسی با ۱۰۰ درس و رابط کاربری جذاب.

## ویژگی‌ها
- ۱۰۰ درس هدفمند از مبتدی تا پیشرفته
- صفحه‌کلید فارسی تعاملی با راهنمای انگشتان
- نمایش سرعت و دقت تایپ
- سیستم ذخیره‌سازی پیشرفت
- رابط کاربری تاریک با انیمیشن‌های جذاب

## نحوه اجرا
فایل `main.exe` را اجرا کرده و از پوشه‌های `fonts/` و تصویر `joke2.jpg` اطمینان حاصل کنید.

## ساخته شده با
- Python 3.11
- PyQt6
- OpenCV
- PyInstaller
